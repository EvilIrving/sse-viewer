# SSE Viewer - SSE 消息专用调试面板

## 🎯 解决什么问题？

Chrome Network 面板**可以看到** SSE 消息，但调试体验很糟糕：

### 😫 现状：繁琐的调试流程

1. **SSE 请求淹没在一堆 Network 请求中** - 需要逐个翻找
2. **每条消息都要手动点击查看** - EventStream 标签来回切换
3. **JSON 数据需要手动复制** - 粘贴到 JSON 格式化工具
4. **多个连接时更加混乱** - 分不清哪条消息属于哪个连接
5. **历史消息刷新就丢失** - 无法回看之前的推送

### ✨ SSE Viewer：一站式解决方案

✅ **专门的 SSE 面板** - 所有 SSE 消息自动汇总到一起  
✅ **自动解析 JSON** - 无需复制粘贴，自动格式化显示  
✅ **按连接分组** - 多个连接清晰区分，一目了然  
✅ **历史记录保留** - 可随时回看之前的消息  
✅ **快速过滤搜索** - 输入关键词立即定位

## 🚀 使用方法

### 安装
从 Chrome Web Store 安装，或点击「添加至 Chrome」按钮

### 使用（3 步）
1. **打开 DevTools** - 按 F12 或右键「检查」
2. **切换到 SSE Viewer 标签** - 在 Network、Console 旁边
3. **刷新页面** - 面板自动捕获所有 SSE 消息

就是这么简单！

### 主要操作
- **查看消息** - 点击任意消息，右侧显示格式化后的 JSON
- **展开连接** - 点击 ▶ 查看该连接的所有历史消息
- **搜索过滤** - 顶部输入关键词快速定位
- **清空面板** - 点击「清空」按钮重新开始
